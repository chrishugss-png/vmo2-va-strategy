STAGE 02 The Master Router via Google Tag Manager URL pattern → entry point → widget script
GTM

The Master Router

Inspects the page URL, matches it against regex patterns, and injects the correct widget script for the customer's journey.

How the match works
if url ~= /retention/
  → inject vm-retain.js
elif url ~= /movers/
  → resolve sub-route
else → vm-sales.js

GTM evaluates the URL hierarchy left-to-right. The first matching regex pattern determines which widget script loads into the DOM — each widget carries its own context payload and routing keys.
